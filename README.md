## 开发准备工作
如果不需要每次commit自动纠正缩进可以忽略这一段

1. 安装`yarn`（如果用homebrew的话可以直接跑`brew install yarn`）
2. clone本repo到本地后先跑`yarn`，安装library（目前还只是一些开发用工具）
3. `yarn husky install`
